Aplicación Web: Modelo OSI y Redes de Computadoras

Esta es una aplicación web educativa que trata sobre redes de computadoras. Utiliza las siguientes tecnologías:

  • Python
  • Flask (framework web)
  • MySQL (base de datos)
  • mysql-connector-python (conector de Python a MySQL)
  • HTML5
  • Redes
  • Protocolos
