¿Qué hace la Capa de Red?

La Capa de Red es la tercera capa del modelo OSI y se encarga del enrutamiento y direccionamiento lógico de los paquetes de datos a través de la red. Esta capa determina la mejor ruta para que los datos lleguen desde el origen hasta el destino, incluso si están en redes diferentes.

Sus funciones principales incluyen el direccionamiento lógico mediante direcciones IP, el enrutamiento de paquetes entre redes, la fragmentación de datos grandes, y el control de congestión de la red.

Protocolos de la Capa de Red

IP (Internet Protocol)

Descripción: Es el protocolo principal de internet que proporciona direccionamiento lógico y enrutamiento de paquetes.

Función: Asigna direcciones IP únicas a cada dispositivo, fragmenta paquetes grandes y determina la ruta de transmisión.

ICMP (Internet Control Message Protocol)

Descripción: Protocolo de control que permite a los dispositivos de red enviar mensajes de error y control.

Función: Proporciona información sobre el estado de la red, errores de transmisión y mensajes de control como ping.

ARP (Address Resolution Protocol)

Descripción: Protocolo que mapea direcciones IP a direcciones MAC en redes locales.

Función: Permite que los dispositivos encuentren la dirección MAC correspondiente a una dirección IP específica.

RIP (Routing Information Protocol)

Descripción: Protocolo de enrutamiento dinámico que permite a los routers compartir información sobre la topología de la red.

Función: Los routers intercambian información sobre las rutas disponibles y calculan las mejores rutas hacia cada destino.

OSPF (Open Shortest Path First)

Descripción: Protocolo de enrutamiento avanzado que calcula las rutas más cortas basándose en el estado de los enlaces.

Función: Proporciona enrutamiento eficiente y convergencia rápida en redes grandes y complejas.

Funcionalidades de Prueba

Como usuario registrado, puedes realizar las siguientes pruebas relacionadas con la Capa de Red:

1. Análisis de Direcciones IP

Esta prueba te permite entender el direccionamiento IP:

  • Convierte direcciones IP entre formato decimal y binario
  • Identifica la clase de red y máscara de subred
  • Calcula direcciones de red y broadcast
  • Analiza rangos de direcciones IP válidas

Protocolos involucrados: IP (IPv4 e IPv6)

Conceptos de red: Direccionamiento IP, subredes, máscaras de red

2. Simulación de Enrutamiento

Experimenta con el proceso de enrutamiento de paquetes:

  • Visualiza tablas de enrutamiento
  • Simula el envío de paquetes entre redes
  • Analiza diferentes algoritmos de enrutamiento
  • Observa el proceso de fragmentación de paquetes

Protocolos involucrados: IP, RIP, OSPF

Conceptos de red: Enrutamiento, tablas de enrutamiento, algoritmos de enrutamiento

3. Herramientas de Diagnóstico de Red

Utiliza herramientas de diagnóstico comunes:

  • Simulación de comando ping (ICMP)
  • Análisis de trazas de ruta (traceroute)
  • Resolución de nombres DNS
  • Verificación de conectividad de red

Protocolos involucrados: ICMP, DNS, ARP

Conceptos de red: Diagnóstico de red, resolución de problemas, conectividad

Ejemplo Práctico: Enrutamiento de un Mensaje

Cuando envías un mensaje desde Higueras a Zarautz, la Capa de Red:

  1. Direccionamiento: Tu mensaje se empaqueta con tu IP origen y la IP destino de tu amigo en España
  2. Enrutamiento local: El paquete se envía al router de tu proveedor de internet en Córdoba
  3. Enrutamiento nacional: El paquete viaja por la red troncal de Argentina hacia Buenos Aires
  4. Enrutamiento internacional: El paquete cruza el océano Atlántico por cables submarinos
  5. Enrutamiento europeo: El paquete llega a España y se enruta hacia el País Vasco
  6. Entrega final: El paquete llega a Zarautz y se entrega a tu amigo

Direcciones IP

Las direcciones IP identifican de manera única cada dispositivo en una red:

Versión Formato Ejemplo Características
IPv4 XXX.XXX.XXX.XXX 192.168.1.100 32 bits, 4 octetos, formato decimal
IPv6 XXXX:XXXX:XXXX:XXXX:XXXX:XXXX:XXXX:XXXX 2001:0db8:85a3:0000:0000:8a2e:0370:7334 128 bits, 8 grupos hexadecimales
