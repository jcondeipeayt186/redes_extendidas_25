¿Qué son las Redes de Computadoras?

Una red de computadoras es un conjunto de dispositivos interconectados que comparten recursos e información.

Un Viaje Digital: Las Higueras → Génova

Imagina que desde tu computadora en Las Higueras, Córdoba, Argentina, envías un mensaje a un celular en Génova, Italia. Este simple acto involucra una compleja red de dispositivos y protocolos que trabajan en conjunto:

  • Tu computadora se conecta a un router WiFi
  • El router se conecta a un proveedor de internet
  • Los datos viajan por cables submarinos y satélites
  • Finalmente llegan al celular en Génova
Ruta de Datos

Argentina → Brasil → Portugal → España → Francia → Italia

Dispositivos de Red

Computadoras

Dispositivos finales que generan y consumen datos

Routers

Dirigen el tráfico entre diferentes redes

Switches

Conectan dispositivos dentro de una misma red

Módems

Convierten señales digitales a analógicas y viceversa

Topologías de Red

Estrella

Todos los dispositivos se conectan a un punto central (hub o switch)

  • Fácil de mantener
  • Un fallo no afecta toda la red
  • Dependiente del punto central
Anillo

Los dispositivos se conectan en forma de círculo

  • Eficiente en el uso del cable
  • Buena para redes pequeñas
  • Un fallo puede romper toda la red
Malla

Cada dispositivo se conecta con todos los demás

  • Máxima redundancia
  • Muy confiable
  • Costosa y compleja

Tipos de Red

LAN (Local Area Network)

Red de área local que conecta dispositivos en un área geográfica limitada.

  • Oficinas y hogares
  • Alta velocidad de transmisión
  • Control total sobre la red
WAN (Wide Area Network)

Red de área amplia que conecta redes LAN a través de grandes distancias.

  • Internet es la WAN más grande
  • Menor velocidad que LAN
  • Requiere proveedores de servicios
MAN (Metropolitan Area Network)

Red de área metropolitana que cubre una ciudad o región.

  • Conecta múltiples LAN
  • Velocidad intermedia
  • Usada por empresas y gobiernos
PAN (Personal Area Network)

Red personal que conecta dispositivos cercanos a una persona.

  • Bluetooth y WiFi Direct
  • Alcance limitado (10 metros)
  • Dispositivos personales

Modelos de Red

Modelo TCP/IP

Modelo de 4 capas desarrollado por el Departamento de Defensa de EE.UU.

  1. Capa de Aplicación
  2. Capa de Transporte
  3. Capa de Internet
  4. Capa de Acceso a Red
Modelo OSI

Modelo de 7 capas desarrollado por ISO para estandarizar las comunicaciones.

  1. Capa Física
  2. Capa de Enlace de Datos
  3. Capa de Red
  4. Capa de Transporte
  5. Capa de Sesión
  6. Capa de Presentación
  7. Capa de Aplicación

Historia de las Redes de Computadoras

1960s - ARPANET

Primera red de computadoras creada por el Departamento de Defensa de EE.UU.

1970s - TCP/IP

Desarrollo de los protocolos TCP/IP que aún se usan hoy en día.

1980s - Modelo OSI

ISO desarrolla el modelo OSI para estandarizar las comunicaciones.

1990s - World Wide Web

Tim Berners-Lee crea la World Wide Web, revolucionando Internet.

2000s - Redes Sociales

Aparición de las redes sociales y la Web 2.0.

2010s - IoT y 5G

Internet de las Cosas y redes 5G transforman la conectividad.
