¿Qué son las Redes de Computadoras?

Una red de computadoras es un conjunto de dispositivos informáticos interconectados que comparten recursos e información. Imagina el increíble viaje que hace un mensaje desde una computadora en Las Higueras, Córdoba, Argentina, hasta un celular en Génova, Italia.

El Viaje de un Mensaje

Cuando envías un mensaje desde Las Higueras a Génova, este viaja a través de:

Red Local (LAN)
Tu computadora se conecta a un router/módem
Proveedor de Internet
El mensaje sale por cables de fibra óptica
Backbone Nacional
Viaja por la infraestructura de Argentina
Cables Submarinos
Cruza el océano Atlántico por cables submarinos
Red Europea
Entra a Europa por España o Portugal
Red Italiana
Viaja por la infraestructura italiana
Red Local Destino
Llega al celular en Génova
¡Increíble! Todo este proceso ocurre en milisegundos, gracias a protocolos y tecnologías que garantizan la entrega confiable de la información.

Dispositivos de Red

Los dispositivos de red son los componentes físicos que permiten la comunicación entre computadoras y otros dispositivos.

Dispositivos de Interconexión
  • Router: Conecta diferentes redes y dirige el tráfico
  • Switch: Conecta dispositivos en una red local
  • Hub: Dispositivo básico que repite señales (obsoleto)
  • Bridge: Conecta segmentos de red
  • Gateway: Conecta redes con diferentes protocolos
Dispositivos Finales
  • Computadoras: PCs, laptops, servidores
  • Dispositivos Móviles: Smartphones, tablets
  • Impresoras de Red: Compartidas en la red
  • IoT: Sensores, cámaras, dispositivos inteligentes
Medios de Transmisión
Cable UTP
Cable de par trenzado (Ethernet)
Fibra Óptica
Transmisión por luz (alta velocidad)
WiFi
Comunicación inalámbrica
Satélites
Comunicación por satélite

Topologías de Red

La topología de red define la forma física o lógica en que están conectados los dispositivos de una red.

Topologías Físicas
  • Bus: Todos los dispositivos conectados a un cable común
  • Estrella: Todos conectados a un dispositivo central (switch)
  • Anillo: Dispositivos conectados en forma circular
  • Malla: Cada dispositivo conectado a todos los demás
  • Árbol: Jerarquía de dispositivos (como ramas de un árbol)
  • Híbrida: Combinación de diferentes topologías
Topologías Lógicas
  • Broadcast: Un dispositivo envía a todos los demás
  • Point-to-Point: Comunicación directa entre dos dispositivos
  • Multicast: Envío a un grupo específico de dispositivos

Tipos de Redes

PAN (Personal Area Network)
Red personal (Bluetooth, USB)
LAN (Local Area Network)
Red local (casa, oficina, escuela)
MAN (Metropolitan Area Network)
Red metropolitana (ciudad)
WAN (Wide Area Network)
Red de área amplia (países, continentes)

  • Redes Privadas: De uso exclusivo de una organización
  • Redes Públicas: Accesibles para cualquier usuario
  • Redes Híbridas: Combinación de privadas y públicas

  • Cliente-Servidor: Servidores centralizados que atienden clientes
  • P2P (Peer-to-Peer): Todos los dispositivos son iguales
  • Cloud Computing: Recursos distribuidos en la nube

Modelos de Referencia

Modelo TCP/IP

El modelo TCP/IP es el estándar actual de Internet, desarrollado por DARPA en los años 70.

  1. Capa de Aplicación: HTTP, FTP, SMTP, DNS, SSH
  2. Capa de Transporte: TCP (confiable), UDP (rápido)
  3. Capa de Internet: IP, ICMP, ARP
  4. Capa de Acceso a Red: Ethernet, WiFi, PPP
Modelo OSI

El modelo OSI es un estándar conceptual desarrollado por ISO en 1984.

  1. Capa Física: Transmisión de bits por el medio
  2. Capa de Enlace de Datos: Control de errores y acceso al medio
  3. Capa de Red: Enrutamiento y direccionamiento lógico
  4. Capa de Transporte: Control de flujo y confiabilidad
  5. Capa de Sesión: Gestión de sesiones entre aplicaciones
  6. Capa de Presentación: Formato y encriptación de datos
  7. Capa de Aplicación: Interfaz con el usuario

Historia de las Redes de Computadoras

Década de 1960
  • 1969: ARPANET - Primera red de conmutación de paquetes
  • 1960s: Desarrollo de conceptos de conmutación de paquetes
Década de 1970
  • 1973: Desarrollo del protocolo TCP/IP
  • 1974: Primera especificación de Ethernet
  • 1977: Primera demostración de ARPANET
Década de 1980
  • 1983: ARPANET adopta TCP/IP
  • 1984: Introducción del modelo OSI
  • 1985: NSFNET conecta universidades estadounidenses
  • 1989: Nacimiento de la World Wide Web
Década de 1990
  • 1991: ARPANET se convierte en Internet
  • 1993: Navegador Mosaic
  • 1995: Comercialización de Internet
  • 1998: Google es fundado
