Capa 3 - Red

Enrutamiento y Direccionamiento Lógico

La capa de red se encarga del enrutamiento de paquetes entre redes diferentes y proporciona direccionamiento lógico para identificar dispositivos.

¿Qué es la Capa de Red?

La Capa 3 del modelo OSI se encarga del enrutamiento de paquetes entre redes diferentes y proporciona direccionamiento lógico.

Función Principal: Determinar la mejor ruta para enviar paquetes de datos entre redes diferentes.

Responsabilidades de la Capa de Red

Enrutamiento
Determinar la mejor ruta para los paquetes
Direccionamiento lógico
Asignar direcciones IP a dispositivos
Fragmentación
Dividir paquetes grandes en fragmentos
Control de congestión
Gestionar el tráfico de red
Calidad de servicio
Priorizar diferentes tipos de tráfico
Interconexión de redes
Conectar redes diferentes

Protocolos de la Capa 3

IP (Internet Protocol)

Protocolo principal de Internet:

  • IPv4: 32 bits de direccionamiento
  • IPv6: 128 bits de direccionamiento
  • No orientado a conexión
  • No garantiza entrega
ICMP

Internet Control Message Protocol:

  • Mensajes de control y error
  • Ping y traceroute
  • Información de estado
  • Diagnóstico de red
ARP

Address Resolution Protocol:

  • Mapeo IP a MAC
  • Tabla ARP
  • Broadcast para resolución
  • Cache local
Protocolos de Enrutamiento

Protocolos para intercambiar información de rutas:

  • RIP (Routing Information Protocol)
  • OSPF (Open Shortest Path First)
  • BGP (Border Gateway Protocol)
  • EIGRP (Enhanced Interior Gateway Routing)
