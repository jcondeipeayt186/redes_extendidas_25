Información sobre las Capas del Modelo OSI

Capa 1 - Física

Transmisión de bits por el medio físico (cables, ondas de radio, etc.)

Capa 2 - Enlace de Datos

Control de errores y acceso al medio físico

Capa 3 - Red

Enrutamiento y direccionamiento lógico entre redes

Capa 4 - Transporte

Control de flujo y confiabilidad en la entrega de datos

Capa 5 - Sesión

Gestión de sesiones entre aplicaciones

Capa 6 - Presentación

Formato y encriptación de datos

Capa 7 - Aplicación

Interfaz con el usuario y servicios de red
